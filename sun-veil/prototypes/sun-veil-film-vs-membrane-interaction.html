Sun Veil PDP / Film vs Membrane · Interaction Demo

SVG 시각 증명에 인터랙션 적용 — 두께 비교 + 공기 통과.

두 시각 증명에 절제된 인터랙션을 적용한 작동 시안. 두께 비교는 한 번의 stagger draw로 데이터 임팩트, 공기 통과는 무한 loop로 영상 대체. 페이지 진입 시 자동 작동, 데스크탑 호버 시 강조.

두께 비교

머리카락 0.07mm → 비닐랩 0.012mm → Sun Veil 0.004mm. 진입 시 한 번 그려지고 멈춤.

공기 통과 비교

필름은 정체, 멤브레인은 통과. 항상 작동하는 무한 loop. 멤브레인 영역에 호버하면 흐름 가속 (데스크탑).

Notes

두께 비교 — 진입 시 stagger draw + 수치 count-up + Sun Veil 막대 pulse 한 번. 데스크탑은 row 호버 시 Sun Veil 막대 강조 + 비율 라벨. "다시 보기" 버튼으로 모션 재현.

공기 통과 — 멤브레인 입자가 위→다공성→아래로 무한 loop, 필름 입자는 막혀서 미세 진동. 다공성 구멍은 미세 fade pulse. 데스크탑 호버 시 흐름 가속(animation-duration 4.2s → 2.4s).

접근성 — prefers-reduced-motion: reduce 시 모든 모션 비활성화, SVG는 정적 표시.
