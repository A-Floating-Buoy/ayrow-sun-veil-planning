편집 결정 / 논의 필요 항목

편집 방식에 대해 한 번 더 정렬이 필요한 핵심 결정 8개. 각 카드는 — What 변경 내용 / Why 사유 / To discuss 논의 포인트.

D1 · Section Order (W1)

Film vs Membrane = 02번 코어 즉시 노출

Why Membrane은 현재 시장 단독 우위 차별 요소(경쟁 제품군이 Film형). 이 코어 메시지가 5번째 섹션에 묻히면 사용자가 안 읽고 이탈할 위험. Hero 직후 02번에 두어 코어 즉시 노출. 02 섹션은 페이지에서 가장 큰 시각·서사 비중. Quick FAQ는 코어 직후 03번.

결정 완료: W1으로 확정. 02 섹션은 비교 표 + "숨을 못 쉬기 때문입니다" + 5개 불편 + "필름막 버리고 나노 섬유 선택" + Closing 흐름. Membrane 우위 직접 자랑 X — 기존 Film 문제 해결로 우회.

D2 · Section New

Invisible Demonstration을 단독 섹션(03)으로 신설

Why 원문 07 KEY EXPERIENCE의 Invisible Veil이 텍스트로만 처리됨 → 시각 demo로 승격해야 NO 반전의 페이오프가 살아난다.

논의: 시각 자산(메이크업 위 부착 / 측면 thickness) 확보 가능 시점. 자산 미확보 시 인포그래픽으로 갈지 섹션 자체를 보류할지.

D3 · Section Merge

Why This Zone을 When + Where 2-Part로 통합

Why 원문 03-2의 "이런 순간"(When)과 광대존 통찰(Where)이 다른 섹션처럼 다뤄지면 같은 섹션의 두 통찰이 흩어진다. 통합으로 인지 순서(자기 상황 → 부위) 확보.

논의: 예민 피부 강조 박스가 Part 1 안에 있는 게 맞는지, 별도 톤(빨간 표시)이 적절한지.

D4 · Section Reframe

Use Cases를 시간축 리프레임(B안)으로

Why 7개 평등 카드 = 자산 7장 부담. "흐름이 끊기지 않는 보호"(Seamless) 메시지와도 직결되지 않음. 시간축으로 묶으면 자산 1-2장으로도 메시지가 더 강해진다.

논의: 시간 매핑이 임의값(06:30 / 10:00 / 11:00 / 14:00 / 16:00 / 18:00). 고객사·브랜드 협의로 조정 필요.

D5 · Copy Constraint

Reverse AG Code™ — 효능 단정 카피 전부 미노출

Why 본 제품은 공산품으로 등재. 화장품에 허용되는 효능 표현(NAD⁺ / SIRT1 / DNA 수선 / 브라이트닝 등)은 표기 충돌 가능. 섹션 정체성을 효능 어필 → 기술 정체성·구성 공개로 전환.

논의: "피부 컨디션을 고려한 포뮬러" 같은 우회 표현도 같이 RA 검수. 섹션 위치를 단독(현 08)으로 둘지 How It Works Layer 2로 흡수할지.
