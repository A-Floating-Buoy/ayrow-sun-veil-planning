두 SVG에 동일 수준 인터랙션 — Layer Split + Moisture-Bonding.

Layer Split은 처음에 두 레이어가 겹쳐 있다가 분리되며 등장. Moisture-Bonding은 곡선이 좌→우로 그려진 후 수분 입자가 차례로 등장하고 미세 shimmer 무한 loop. 두 SVG 모두 진입 시 자동 작동, "다시 보기"로 재현.

Layer Split — 두 레이어가 분리되며 등장

처음에는 두 레이어가 겹쳐 있는 상태(실제 제품). 진입 시 위 레이어가 위로 8px, 아래 레이어가 아래로 12px translate + fade-in으로 분리되며 등장. 다공성 점들이 stagger pop, 분리 화살표가 마지막 등장.

Moisture-Bonding — 곡선이 좌→우로 그려지고 수분이 자리잡음

피부 곡선이 먼저 좌→우 stroke 그려짐 → 멤브레인 곡선이 600ms 후 같은 방향으로 그려짐 → 두 곡선 사이에 수분 입자가 차례로 등장 → 수분이 미세 shimmer로 무한 진동 ("표면 장력으로 잡혀있다").

인터랙션 검증 포인트

Layer Split — 진입: 처음 두 레이어가 겹쳐서 안 보이다가, 위 레이어가 위에서 ↓, 아래 레이어가 아래에서 ↑로 분리되며 등장. 200ms stagger로 순차.

Layer Split — 디테일: 다공성 점 8개가 60ms 간격 stagger pop (cubic-bezier overshoot). 분리 화살표가 1.5s에 fade-in. 라벨은 200ms stagger.

Layer Split — 호버: 데스크탑에서 SVG 호버 시 분리 폭이 더 커짐 (위 -6px, 아래 +6px). "더 자세히 보고 싶다"에 응답.

Moisture-Bonding — 진입: 피부 곡선이 1.5s 동안 좌→우 stroke-dashoffset 그려짐. 600ms 후 멤브레인 곡선이 같은 방향으로 그려짐. 1.8s부터 수분 입자 8개가 100ms stagger pop.

Moisture-Bonding — 무한 loop: 수분 등장 1초 후부터 미세 shimmer (translateY ±1.5px, 3.6s cycle). 입자별 200ms stagger로 자연스러운 비동기 진동.

Moisture-Bonding — 호버: 데스크탑에서 SVG 호버 시 수분 opacity 0.65 → 0.95, shimmer cycle 3.6s → 1.8s 가속.

접근성: prefers-reduced-motion 시 모든 transition·animation 비활성, 정적 표기.
