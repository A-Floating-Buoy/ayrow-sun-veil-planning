스크롤하면 차트가 가로로 진행하며 두께가 얇아진다.

섹션이 화면 가득 펼쳐지고, 페이지 세로 스크롤이 곧 차트의 가로 panning이 됩니다. 차트는 화면 너비의 약 2.8배. 신용카드부터 Sun Veil(약 1px)까지 흐름으로 줄어드는 동안 친숙한 사물 마커가 viewport에 진입할 때마다 라벨이 등장하고, 마지막에 Sun Veil이 임팩트로 자리잡습니다.

인터랙션 검증 포인트

Full-bleed gradient — 섹션 전체가 Bleu vertical gradient (deep → bleu → soft). 차트 영역은 흰색 그라디언트 + 흰 stroke으로 대비.

Sticky horizontal pan — 섹션 height 360vh, sticky inner 100vh. 페이지 세로 스크롤 progress 0→1이 차트 translateX 0 → -(280vw - 100vw) = -180vw로 매핑.

마커 진입 활성 — 각 마커의 screen X가 viewport 0 ~ 우측 -100px 사이에 진입하면 fade-in (한 번, cumulative). 사용자가 중간에 빠르게 스크롤해도 모든 마커가 차례로 등장.

Sun Veil 강조 — 마지막 마커는 Bleu-deep 박스 + 흰 텍스트 + 큰 사이즈 + scale 0.9 → 1.05 → 1.0 pop. 차트 우측 끝에서 등장.

상단 progress 바 — 화면 우상단에 얇은 흰 바 + % 라벨. 사용자가 현재 어디에 있는지 시각적 피드백.

접근성 — prefers-reduced-motion 시 sticky 해제, chart width 100%로 정적 표시, 모든 마커 즉시 노출.

모바일 — chart width 320vw로 늘리고 (좁은 화면 보상), 라벨 박스 텍스트 크기 축소. progress 인디케이터는 head 영역 세로 정렬.
