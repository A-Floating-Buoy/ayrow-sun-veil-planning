눈가·광대처럼 특정 부위가 유독 노출될 때

디자인 노트

  • 레이아웃 — 데스크탑 3열, 태블릿 2열, 모바일 1열. gap: 16px
  • 아이콘 — 단순 라인 (1.4px stroke) + currentColor + Bleu 액센트로 핵심 강조 ①덧바르기 X / ③정점 태양 / ④햇살 / ⑤햇살 / ⑥광대존
  • 인터랙션 — 진입 시 80ms stagger fade-up, 데스크탑 호버 시 -4px lift + 미세 box-shadow + 테두리 Bleu tint
  • 카피 강조 — 카피의 핵심 phrase에 <strong>으로 시각 위계 (사용자가 카드를 빠르게 훑을 때 핵심만 잡힘)
  • 접근성 — 각 SVG에 role="img" + aria-label로 스크린리더 지원, prefers-reduced-motion fallback

카피 vs 시각 매핑

이전 분석에서 카피의 모호한 단어(① "힘든 이유", ③ "흐름", ⑤ "생활 자외선", ⑥ "노출")가 즉시 이해되지 않는 약점이 있었음. 아이콘이 그 모호함을 시각으로 보강:

  • ① 선크림 튜브 + ✕ — "덧바름" 자체가 제한된 상황 명시
  • ② 손가락 + 메이크업 표면 + 멈춤 점선 — 행동의 망설임 시각화
  • ③ 태양 궤적 (오전→정오→오후) — "하루 종일 흐름" 구체화
  • ④ 러닝 사람 + 햇살 — 활동 + 환경
  • ⑤ 창문 + 햇살 들어옴 — "생활 자외선" = 창가 환경 명시
  • ⑥ 얼굴 + 광대 점선 — 부위 특이성 시각화

순서 변형 옵션

현재는 원문 순서. 만약 명확한 카피를 앞으로 두고 추상 카피를 뒤로 두려면:

② → ④ → ⑥ → ① → ⑤ → ③ (명확 → 추상)

이 순서면 사용자가 첫 두 카드(메이크업 위·러닝)에서 즉시 공감 모드에 들어가고, 그 흐름에 추상 카피가 따라와 더 잘 받아들여짐. 단 원문의 의도된 순서가 있다면 유지.
