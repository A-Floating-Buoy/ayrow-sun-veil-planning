Sun Veil PDP / Thickness Experiments · Interactive

4가지 시각화 + 동일 인터랙션 — 메타포 비교.

각 안에 메타포에 맞는 stagger draw 모션 + 수치 count-up + Sun Veil 강조 pulse를 적용. 진입 시 자동 작동, "다시 보기" 버튼으로 재현 가능.

안 A는 scaleX(데이터 차트 메타포), B는 scaleY(베이스라인 위로 솟음), C는 scale(실물 단면 등장), D는 scaleX(피부 위 적층), E는 scale(메인 강조). 모두 동일 280ms stagger.

Experiment A · Baseline

현재 안 — 가로 긴 막대 (scaleX 좌→우)

막대가 좌→우로 길게 그려지는 데이터 시각화 정석. 진입 시 머리카락 → 비닐랩 → Sun Veil 순서로 stagger draw + 수치 count-up + Sun Veil pulse.

강점

데이터 시각화 정석. stagger draw 인터랙션이 자연스럽게 어울림.

약점

막대 = "길이" 메타포. 같은 길이 3개를 보면 시선이 길이 차이부터 찾음.

Experiment B · Cross-Section

짧은 단면 가로 배치 — scaleY (베이스라인에서 위로 솟음)

같은 베이스라인 위에 단면이 위로 솟아오르는 모션. "두께가 자란다"의 시각.

강점

길이 변수 짧고 동일. 시선이 두께(세로)로 집중. scaleY 모션이 "두께가 자란다"의 의미와 일치.

약점

Sun Veil(2.9px)이 너무 얇아 거의 선처럼 보임.

Experiment C · Real-Life Metaphor

실물 단면 — 원은 scale 등장, 띠는 fade

머리카락 원형 단면은 scale 0→1로 등장(현미경 줌 메타포), 띠는 자연스러운 fade.

강점

원 scale 등장이 "현미경 줌" 메타포와 어울림. 형태 다양성으로 시각 흥미.

약점

머리카락만 원형, 다른 둘은 띠 — 시각 일관성 부족.

Experiment D · Stacked on Skin

피부 위 적층 — scaleX (피부 위에 얹히는 메타포)

피부 표면 위에 세 항목이 차례로 얹히는 시각. scaleX로 좌→우 그려지면서 "올려진다"의 모션.

강점

"피부 위 얹힘" 제품 본질 메타포. scaleX 모션이 "올려진다" 의미와 일치.

약점

가로 폭 120px이 여전히 길어 길이 인지 일부 잔존.

Experiment E · Magnified Detail

확대 단면 — 메인 scale 등장 + 작은 비교 fade

Sun Veil 확대 단면이 scale로 메인 등장 (임팩트), 작은 비교 카드들은 fade로 따라 등장. 메시지 위계 명확.

강점

Sun Veil 메인 자리 + scale 등장으로 임팩트 집중. "800배 확대" 변환이 두께 감각을 구체화.

약점

"800배 확대" 개념 이해 비용. 비교 카드 3개 추가가 다소 많음.

Discussion

같은 데이터 + 다른 메타포 + 메타포에 맞는 모션. 진입 시 자동 작동, "다시 보기" 버튼으로 재현. 모든 안에 동일한 280ms stagger와 Sun Veil pulse 적용.

안 비교 후 선호 안 알려주시면, 그 안에 디테일(hover 강조, 추가 caption fade 등) 보강 → B-5 wireframe 03 섹션에 통합.
