두께가 흐름으로 줄어드는 영역 차트.

X축 progress 0% → 100%, Y축 두께. 영역이 좌→우로 부드럽게 채워지며 곡선이 점진적으로 얇아짐. 친숙한 사물(신용카드 → 종이 → 머리카락 → 비닐랩) 마커가 시점에 도달할 때 라벨 박스로 등장하고, 마지막 우측 끝에 Sun Veil이 강조 등장.

인터랙션 검증 포인트

영역 reveal — 6.2초 동안 좌→우 clip-path inset(0 100% 0 0) → inset(0). cubic-bezier(0.5, 0, 0.5, 1)로 시작·끝 빠르고 중간 균일.

외곽선 stroke — 영역과 동기로 stroke-dashoffset 2400 → 0. 영역과 같이 좌→우 그려지는 시각.

마커 stagger — 영역 reveal 시작 후 1.0s, 2.0s, 3.0s, 4.0s, 5.0s 시점에 마커가 차례로 fade-in. 영역 곡선이 그 X 위치에 도달하는 시점과 동기.

마지막 Sun Veil 마커 — Bleu 배경 박스 + 큰 흰색 텍스트 + 등장 시 scale 0.85 → 1.06 → 1.0 미세 overshoot pop.

우측 mini-legend — 항상 표시, Sun Veil은 기본 강조. 추후 마커 도달 시 해당 legend 아이템도 강조하는 동기 가능.

접근성 — prefers-reduced-motion 시 영역과 마커 즉시 표기.
